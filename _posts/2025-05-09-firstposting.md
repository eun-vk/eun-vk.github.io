#파이썬 리스트 해부학: 인덱싱과 슬라이싱 편

## 📚리스트(list)의 의미와 특징   
📌**리스트란(list)?**  
다양한 자료형의 데이터를 하나의 변수에 저장하는 하나의 목록이다.  


📌**리스트의 특징**  

- 숫자형, 문자형, bool형 등 여러 자료를 리스트에 담을 수 있다.  
',(쉼표)'로 구분한다.  
```
s = [5, "사과", "바나나", True, 45]
```
- 리스트의 인덱스는 0부터 시작한다. 
```
s = [5, "사과", "바나나", True, 45] 

* 각 요소의 인덱스는 다음과 같다. 
| 5 -> 0 | "사과" -> 1 | "바나나" -> 2 | True -> 3 | 45 -> 4 |
```
- 리스트는 왼쪽에서 오른쪽으로 요소가 추가되며, pop()을 사용하면 스택(후입선출) 방식처럼 사용할 수 있다.
```
s = [5, "사과", "바나나", True, 45] 

위 리스트의 각 요소 입력 순서는 왼쪽에서 오른쪽이다. 
만약 pop()을 사용해 항목을 추출한다면 가장 나중에 입력된 45가 리스트에서 추출되어 삭제된다. 
s.pop() 
print(s)

[실행결과]
[5, "사과", "바나나", True]
```
- 리스트 안에 있는 값을 바꿀 수 있다.  
```
fruits = ["사과","오렌지","바나나","키위"]

# fruits 리스트 안에 있는 값 바꾸기.
fruits[2] ="수박"
print(fruits)

[실행결과]
["사과","오렌지","수박","키위"]
```
- 문자열의 특정 문자는 바꿀 수 없다.  
```
s = [5, "사과", "바나나", True, "dog"] 
s[4][0] = "l"  (X)
의도한 것은 dog -> log로 바뀌는 것이지만, 문자열의 특정 문자는 바꿀 수 없다. 
```
- 중복과 중첩을 허용한다. 
```
a = [5, ["사과", "바나나"], True, "dog"]
b = [45, 26, ["토끼","고래",["책", "만화"],"고래"],28]

a는 2중첩, b는 3중첩의 형태이다. 

Q1. a 리스트에서 "사과"를 추출하시오.
a[1][0]
Q2. b 리스트에서 "만화"를 추출하시오.
b[2][2][1]

[2번 문제 해설]
- b[2]= ["토끼", "고래", ["책", "만화"], "고래"]
- b[2][2]= ["책", "만화"]
- b[2][2][1]= "만화"
```
## 📚리스트의 인덱싱(Indexing) 
📌**인덱스(Index)란?**  <span style="color:green">— 각 요소의 "위치"   </span>  
라틴어 '인디카레(indicare)'에서 유래되었으며, '가리키다'라는 뜻을 가지고 있다. 인덱스는  각 요소의 위치(번호)를 말한다. 

📌**인덱싱(Indexing)이란?** <span style="color:green">— 하나의 요소를 "뽑아내는 작업"   </span>  
인덱스{각 요소의 위치(번호)}를 사용해 특정 값을 추출하는 것을 말한다. 즉, 인덱스 번호를 이용해 원하는 요소를 찾아내는 작업을 인덱싱(Indexing)이라고 한다. 
- 인덱스는 0부터 시작한다. 
- [-1]은 마지막 요소를 추출한다. 
![Example Image](https://i.imgur.com/47bSwgP.jpeg)

## 📚리스트의 슬라이싱(Slicing)
📌**슬라이싱이란?** <span style="color:green">— 시퀀스를 "조각내는 기술"     </span>   
슬라이스(Slice)는 '조각'이라는 뜻으로 슬라이싱(Slicing)은 '조각내는 것'을 의미한다. 즉, 시퀀스형 자료(리스트, 문자열, 튜플 등)에서 일부 구간을 조각내 추출하여 '새로운 시퀀스'를 만드는 것을 말한다. 

📌**슬라이싱의 기본 구조**[start:stop:step]  
- 슬라이싱은 ':'로 구분한다. 
- 기본 구조는 처음(start), 끝(stop), 간격(step)으로 구성된다.
- 구조는 아래와 같이 변경 가능하다. 
![Example Image](https://i.imgur.com/p4omZ3l.jpeg)

📌**슬라이싱 사용방법**
- 슬라이싱할 리스트 변수 뒤에 위에서 언급한 5가지의 구조를 활용하여 슬라이싱할 수 있다.   
- 슬라이싱에서  종료(stop)인덱스는 포함하지 않고 그 전까지의 인덱스까지만 포함한다. (중요!🌟🌟🌟)

## 📚예제 문제 풀어보기  

📌**리스트 인덱싱**  
**1.다음 리스트에서 "포도"를 추출하여 출력하시오.**
``` 
fruits = ['사과', '바나나', '포도', '수박']
```
출력결과:
```
포도
```
**2.다음 문제의 출력 결과는?**
```
numbers = [10, 20, 30, 40, 50]
print(numbers[-1])
```
출력결과:
```
50
```
**3.다음 중첩 리스트에서 숫자 5를 추출하여 출력하시오.**
```
data = [[1, 2], [3, 4], [5, 6]]
```
출력결과:
```
5
```
**4. 다음 중첩 리스트에서 40과 "책"을 추출하시오.**
```
data = [[10, 20, ["사과", "바나나"], 40], [30, ["토끼", "고래", ["책", "만화"], "고래"], 50]]
```
출력결과:
```
[40, '책']
```
📌**리스트 슬라이싱**
**1. 다음 리스트에서 '바나나'와 '수박'만 추출하여 출력하시오.**
```
fruits = ['사과', '바나나', '수박', '포도']
```
출력결과:
```
['바나나', '수박']
```
**2. 다음 리스트에서 짝수 인덱스에 해당하는 값들만 추출하시오.**
```
nums = [0, 10, 20, 30, 40, 50, 60]
```
출력결과:
```
[0, 20, 40, 60]
```
**3. 다음 리스트에서 뒤에서 3개 요소만 추출하시오.**
```
animals = ['개', '고양이', '호랑이', '토끼', '말']
```
출력결과:
```
['호랑이', '토끼', '말']
```
**4. 다음 문자열에서 'hello'만 슬라이싱으로 추출하시오.**
```
text = "hello_python_world"
```
출력결과:
```
hello
```
