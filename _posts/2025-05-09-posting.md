<h2 class="text-2xl font-bold mt-4 mb-4">📚리스트(list)의 의미와 특징</h2>
<h3 class="text-xl font-bold mt-4 mb-4">
  📌<strong class="font-bold">리스트란(list)?</strong>
</h3>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 다양한 자료형의 데이터를 하나의 변수에 저장하는 하나의 목록이다.
</h4>
<h3 class="text-xl font-bold mt-4 mb-4">
  📌<strong class="font-bold">리스트의 특징</strong>
</h3>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 숫자형, 문자형, bool형 등 여러 자료를 리스트에 담을 수 있다.
</h4>
<h4 class="text-lg font-bold mt-4 mb-4">• &#39;,(쉼표)&#39;로 구분한다.</h4>
<p class="text-lg my-6">
  <span style="color:green"
    >s = [5, &quot;사과&quot;, &quot;바나나&quot;, True, 45]
  </span>
</p>
<h4 class="text-lg font-bold mt-4 mb-4">• 리스트의 인덱스는 0부터 시작한다.</h4>
<p class="text-lg my-6">
  s = [5, &quot;사과&quot;, &quot;바나나&quot;, True, 45]<br />| 5 -&gt; 0 |
  &quot;사과&quot; -&gt; 1 | &quot;바나나&quot; -&gt; 2 | True -&gt; 3 | 45
  -&gt; 4 |
</p>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 리스트는 왼쪽에서 오른쪽으로 요소가 추가되며, pop()을 사용하면
  스택(후입선출) 방식처럼 사용할 수 있다.
</h4>
<p class="text-lg my-6">
  <span style="color:green">
    s = [5, &quot;사과&quot;, &quot;바나나&quot;, True, 45]
  </span>
  <span style="color:green"><br />s.pop()<br />print(s) </span>
</p>
<p class="text-lg my-6">
  <span style="color:orange"
    >[실행결과]<br />[5, &quot;사과&quot;, &quot;바나나&quot;, True]
  </span>
</p>
<p class="text-lg my-6">
  위 리스트의 각 요소 입력 순서는 왼쪽에서 오른쪽이다. 만약 pop()을 사용해
  항목을 추출한다면 가장 나중에 입력된 45가 리스트에서 추출되어 삭제된다.
</p>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 리스트 안에 있는 값을 바꿀 수 있다.
</h4>
<p class="text-lg my-6">
  <span style="color:green">
    fruits =
    [&quot;사과&quot;,&quot;오렌지&quot;,&quot;바나나&quot;,&quot;키위&quot;] </span
  ><br /><span style="color:green"
    >fruits[2] =&quot;수박&quot; print(fruits)</span
  >
</p>
<p class="text-lg my-6">
  <span style="color:orange"
    >[실행결과]
    [&quot;사과&quot;,&quot;오렌지&quot;,&quot;수박&quot;,&quot;키위&quot;]
  </span>
</p>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 문자열의 특정 문자는 바꿀 수 없다.
</h4>
<p class="text-lg my-6">
  <span style="color:green">
    s = [5, &quot;사과&quot;, &quot;바나나&quot;, True, &quot;dog&quot;] </span
  ><br /><span style="color:orange"> s[4][0] = &quot;l&quot; (X)</span>
</p>
<p class="text-lg my-6">
  의도한 것은 dog -&gt; log로 바뀌는 것이지만, 문자열의 특정 문자는 바꿀 수
  없다.
</p>
<h4 class="text-lg font-bold mt-4 mb-4">• 중복과 중첩을 허용한다.</h4>
<p class="text-lg my-6">
  <span style="color:green">
    a = [5, [&quot;사과&quot;, &quot;바나나&quot;], True, &quot;dog&quot;]</span
  ><br /><span style="color:orange">
    b = [45, 26, [&quot;토끼&quot;,&quot;고래&quot;,[&quot;책&quot;,
    &quot;만화&quot;],&quot;고래&quot;],28]</span
  >
</p>
<p class="text-lg my-6">a는 2중첩, b는 3중첩의 형태이다.</p>
<p class="text-lg my-6">
  <span style="color:pink">
    Q1. a 리스트에서 &quot;사과&quot;를 추출하시오. </span
  ><br />a[1][0]<br /><span style="color:pink">
    Q2. b 리스트에서 &quot;만화&quot;를 추출하시오. </span
  ><br />a[1][0]<br />b[2][2][1]
</p>
<p class="text-lg my-6">[2번 문제 해설]</p>
<ul class="list-disc text-lg list-inside my-6">
  <li class="my-1">
    b[2]= [&quot;토끼&quot;, &quot;고래&quot;, [&quot;책&quot;,
    &quot;만화&quot;], &quot;고래&quot;]
  </li>
  <li class="my-1">b[2][2]= [&quot;책&quot;, &quot;만화&quot;]</li>
  <li class="my-1">b[2][2][1]= &quot;만화&quot;</li>
</ul>
<h2 class="text-2xl font-bold mt-4 mb-4">📚리스트의 인덱싱(Indexing)</h2>
<img src="https://i.imgur.com/47bSwgP.jpeg" width="400px" />

<h3 class="text-xl font-bold mt-4 mb-4">
  📌<strong class="font-bold">인덱스(Index)란?</strong>
  <span style="color:green">— 각 요소의 &quot;위치&quot; </span>
</h3>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 라틴어 &#39;인디카레(indicare)&#39;에서 유래되었으며, &#39;가리키다&#39;라는
  뜻을 가지고 있다. 인덱스는 각 요소의 위치(번호)를 말한다.
</h4>
<h3 class="text-xl font-bold mt-4 mb-4">
  📌<strong class="font-bold">인덱싱(Indexing)이란?</strong>
  <span style="color:green">— 하나의 요소를 &quot;뽑아내는 작업&quot; </span>
</h3>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 인덱스{각 요소의 위치(번호)}를 사용해 특정 값을 추출하는 것을 말한다. 즉,
  인덱스 번호를 이용해 원하는 요소를 찾아내는 작업을 인덱싱(Indexing)이라고
  한다.
</h4>
<h4 class="text-lg font-bold mt-4 mb-4">• 인덱스는 0부터 시작한다.</h4>
<h4 class="text-lg font-bold mt-4 mb-4">• [-1]은 마지막 요소를 추출한다.</h4>
<h2 class="text-2xl font-bold mt-4 mb-4">📚리스트의 슬라이싱(Slicing)</h2>
<h3 class="text-xl font-bold mt-4 mb-4">
  📌<strong class="font-bold">슬라이싱이란?</strong>
  <span style="color:green">— 시퀀스를 &quot;조각내는 기술&quot; </span>
</h3>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 슬라이스(Slice)는 &#39;조각&#39;이라는 뜻으로 슬라이싱(Slicing)은
  &#39;조각내는 것&#39;을 의미한다. 즉, 시퀀스형 자료(리스트, 문자열, 튜플
  등)에서 일부 구간을 조각내 추출하여 &#39;새로운 시퀀스&#39;를 만드는 것을
  말한다.
</h4>
<img src="https://i.imgur.com/p4omZ3l.jpeg" width="400px" />

<h3 class="text-xl font-bold mt-4 mb-4">
  📌<strong class="font-bold">슬라이싱의 기본 구조</strong>[start:stop:step]
</h3>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 슬라이싱은 &#39;:&#39;로 구분한다.
</h4>
<h4 class="text-lg font-bold mt-4 mb-4">
  • 기본 구조는 처음(start), 끝(stop), 간격(step)으로 구성된다.
</h4>
<h4 class="text-lg font-bold mt-4 mb-4">• 구조는 아래와 같이 변경 가능하다.</h4>
<h3 class="text-xl font-bold mt-4 mb-4">
  📌<strong class="font-bold">슬라이싱 사용방법</strong>
</h3>
<img src="https://i.imgur.com/0R7OSp7.jpeg" width="400px" />

<ul class="list-disc text-lg list-inside my-6">
  <li class="my-1">
    슬라이싱할 리스트 변수 뒤에 위에서 언급한 5가지의 구조를 활용하여 슬라이싱할
    수 있다.
  </li>
  <li class="my-1">
    슬라이싱에서 종료(stop)인덱스는 포함하지 않고 그 전까지의 인덱스까지만
    포함한다. (중요!🌟🌟🌟)
  </li>
</ul>
